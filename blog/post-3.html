O Papel do Business Analyst em Empresas de Tecnologia

Muitas pessoas me perguntam o que um Business Analyst faz em uma empresa como o Nubank. A resposta é: depende da empresa, mas o core é sempre o mesmo - traduzir dados em decisões de negócio.

O que Faço no Dia a Dia

  • Análises ad-hoc para liderança executiva
  • Estudos de segmentação de clientes
  • Análises de preço e mercado
  • Monitoramento de KPIs e OKRs
  • Desenvolvimento de dashboards

Habilidades que Importam

Habilidades técnicas são importantes, mas o que realmente diferencia um ótimo BA é a capacidade de fazer as perguntas certas. Entender o contexto de negócios é mais valioso do que conhecer cada função SQL.

A Caixa de Ferramentas

No Nubank, trabalho com Databricks para processamento de dados, Python com Streamlit para dashboards interativos e ocasionalmente Scala para manutenção de banco de dados. Mas a ferramenta mais importante é comunicação clara.

Conselhos para Aspirantes a BA

  • Aprenda SQL profundamente - é a linguagem universal dos dados
  • Entenda o negócio antes de mergulhar nos dados
  • Pratique apresentar insights para pessoas não-técnicas
  • Construa um portfólio de projetos que mostrem impacto
