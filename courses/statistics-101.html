Estatística 101

Fundamentos de estatística para análise de dados

Introdução

Bem-vindo ao Estatística 101! Este curso vai ensinar os conceitos fundamentais de estatística que todo profissional de dados precisa conhecer.

1. O que é Estatística?

Estatística é a ciência de coletar, organizar, analisar e interpretar dados para tomar decisões informadas.

Estatística Descritiva vs. Inferencial

Existem dois ramos principais da estatística:

  • Estatística Descritiva: Resume e descreve dados
  • Estatística Inferencial: Faz previsões baseadas em dados

2. Tipos de Dados

Entender os tipos de dados é fundamental para escolher os métodos estatísticos corretos.

Dados Qualitativos

Dados categóricos que descrevem qualidades ou características (ex: cores, nomes, categorias).

Dados Quantitativos

Dados numéricos que podem ser medidos (ex: altura, peso, temperatura).

3. Medidas de Tendência Central

Medidas de tendência central nos ajudam a entender o valor 'típico' em um conjunto de dados.

  • Média: A média aritmética
  • Mediana: O valor do meio
  • Moda: O valor mais frequente

4. Medidas de Dispersão

Medidas de dispersão nos dizem quão espalhados estão os dados.

5. Noções de Probabilidade

Probabilidade é a base da inferência estatística.

6. Distribuição Normal

A distribuição normal, também conhecida como curva de sino, é um dos conceitos mais importantes em estatística.

7. Teste de Hipóteses

O teste de hipóteses é um método usado para tomar decisões estatísticas usando dados experimentais.

8. Correlação e Regressão

Aprenda a medir relacionamentos entre variáveis e fazer previsões.
