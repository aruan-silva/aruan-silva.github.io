Estatística 101

Fundamentos de estatística para análise de dados

Introdução

Bem-vindo ao Estatística 101! Este curso vai ensinar os conceitos fundamentais de estatística que todo profissional de dados precisa conhecer.

1. O que é Estatística?

Estatística é a ciência de coletar, organizar, analisar e interpretar dados.

Descritiva vs. Inferencial

  • Descritiva: Resume dados
  • Inferencial: Faz previsões

2. Tipos de Dados

Entender os tipos de dados é fundamental para escolher métodos estatísticos.

3. Tendência Central

Média, mediana e moda nos ajudam a entender valores típicos.

4. Dispersão

Medidas de dispersão nos dizem quão espalhados estão os dados.

5. Probabilidade

Probabilidade é a base da inferência estatística.

6. Distribuição Normal

A curva de sino é um dos conceitos mais importantes.

7. Teste de Hipóteses

Tomando decisões estatísticas usando dados.

8. Correlação e Regressão

Medindo relacionamentos e fazendo previsões.
