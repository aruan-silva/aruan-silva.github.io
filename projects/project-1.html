Automação de Pipelines de Dados

Produto de dados que automatiza a criação e análise de pipelines de dados

Python AWS ETL Looker

Contexto

Na Red Ventures Brasil, atuei como Product Owner de um produto de dados projetado para automatizar e analisar pipelines de dados, melhorando a eficiência da equipe de engenharia de dados.

Desafio

A equipe gastava tempo significativo criando e monitorando pipelines de dados manualmente, causando atrasos na entrega de insights para stakeholders.

Solução

  • Desenvolvimento de processos ETL automatizados
  • Criação de dashboards em Looker, Preset e Metabase
  • Implementação de sistemas de monitoramento e alertas
  • Apoio a stakeholders com decisões baseadas em dados

Impacto

  • Redução de 60% no tempo de criação de pipelines
  • Melhoria nas decisões de alocação de orçamento de marketing
  • Otimização de estratégias de posicionamento de produto
  • Melhor gestão da infraestrutura de dados

Tecnologias

Python, AWS (S3, Redshift), Looker, Preset, Metabase, SQL
