Migração para Data Lakehouse

Migração do Amazon Redshift para arquitetura Databricks Delta Lake

Databricks Delta Lake AWS Data Engineering

Contexto

Na Red Ventures Brasil, a infraestrutura de dados era baseada em Amazon Redshift (Data Warehouse). Para melhorar escalabilidade, performance e custo-benefício, decidimos migrar para uma arquitetura de Data Lakehouse.

Arquitetura

  • Armazenamento: AWS S3
  • Processamento: Databricks
  • Formato de Tabelas: Delta Lake
  • Orquestração: Airflow

Meu Papel

Participei das fases de planejamento da migração, validação de dados e testes. Também ajudei a documentar a nova arquitetura e treinar membros da equipe nas novas ferramentas.

Benefícios

  • Transações ACID com Delta Lake
  • Melhor performance para analytics em larga escala
  • Suporte a evolução de schema
  • Capacidades de time travel
  • Batch e streaming unificados

Tecnologias

Databricks, Delta Lake, AWS S3, Amazon Redshift, Python, SQL, Airflow
