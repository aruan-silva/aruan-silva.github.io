Dashboards Executivos

Dashboards de monitoramento de KPIs e OKRs para liderança executiva no Nubank

Databricks Python Streamlit SQL

Contexto

No Nubank, sou responsável por realizar análises ad-hoc para a liderança executiva e desenvolver dashboards que permitem monitoramento em tempo real do desempenho do negócio.

Tipos de Análises

  • Segmentação de clientes
  • Análise de preços
  • Análise de mercado
  • Monitoramento de KPIs e OKRs

Implementação Técnica

Os dashboards são construídos usando Databricks para processamento de dados e Python com Streamlit para visualizações interativas. Também trabalho com manutenção de banco de dados em Scala.

Tecnologias

Databricks, Python, Streamlit, Scala, SQL
