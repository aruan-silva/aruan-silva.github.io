Base de Dados de Balanços Financeiros

Base de dados em Python que expandiu a cobertura de empresas em 10x para análise de equity

Python Google Sheets Finance Automation

Contexto

Na Primus Ventures, a primeira gestora multi-asset class focada em empresas de tecnologia no Brasil, eu era responsável por analisar oportunidades de investimento em ações de SaaS e Cibersegurança.

Desafio

A análise manual de balanços financeiros consumia muito tempo e limitava o número de empresas que podíamos cobrir efetivamente.

Solução

Desenvolvi uma base de dados em Python que coleta, processa e organiza automaticamente balanços financeiros de empresas listadas, integrada com Google Sheets para fácil acesso e análise.

Empresas Analisadas

  • SentinelOne
  • Unity
  • Crowdstrike
  • Okta
  • Tenable
  • GitLab
  • Workday

Resultados

  • Expansão de 10x na cobertura de empresas
  • Redução de 4x no tempo de análise
  • Geração automatizada de relatórios setoriais
  • Melhor gestão dos dados do fundo de ações

Tecnologias

Python, Pandas, Google Sheets API, APIs Financeiras
